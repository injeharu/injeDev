# injeDev

[Minecraft](https://www.minecraft.net/) の **Paper / Spigot** 向け **プラグイン** と、それに付随する **自作ツール**（ビルド補助・CLI など）用のソース置き場です。

**Web サイト（inje Dev の紹介サイト・静的サイト一式）はこのリポジトリには含めません。** サイトはローカルまたは別ホスティングで管理し、このリモートにはプラグイン・ツールのみを載せる前提です。

## ディレクトリ

| パス | 内容 |
|------|------|
| [`plugins/`](plugins/) | Paper / Spigot プラグイン（プロジェクト単位でサブフォルダを増やす想定） |
| [`tools/`](tools/) | サイト以外の開発ツール・スクリプト |

量が増えたらプラグインごとにリポジトリを分割しても問題ありません。

## サイトを公開していた場合（Pages など）

以前このリモートにつないで自動ビルドしていた場合は **リポジトリの中身からサイトがなくなる**ので、ホスティング側で **ソースを差し替える**か、**別リポジトリ・手動デプロイ（Wrangler `pages deploy dist` 等）** に切り替えてください。
