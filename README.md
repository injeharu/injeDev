# injeDev

[Minecraft](https://www.minecraft.net/) の **Paper / Spigot** 向け **プラグイン** と、付随する **自作ツール**（ビルド補助・CLI など）のリポジトリです。

Web サイト（紹介サイト）はこのリポジトリには含めません。

## ディレクトリ

| パス | 内容 |
|------|------|
| [`plugins/`](plugins/) | Paper / Spigot プラグイン（プロジェクト単位でサブフォルダを増やす想定） |
| [`tools/`](tools/) | サイト以外の開発ツール・スクリプト |
